# renjana

A simple greeting page for Renjana is available in `index.html`.

A second matcha café page is available in `cafe.html`.

A third story timeline page is available in `story.html`.

- Open `index.html` in your browser to view the first message and animation.
- Open `cafe.html` for a second page with a cozy matcha café theme.
- Open `story.html` for a third page with a chapter-by-chapter love story.
- Or use GitHub Pages by publishing this repository and setting the site source to `main` branch.
